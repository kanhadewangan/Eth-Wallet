Etheremun Wallet 
 




## Run Locally

Clone the project

```bash
  git clone https://github.com/kanhadewangan/Eth-Wallet.git
```

Go to the project directory

```bash
  cd Eth-Wallet
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
